# Bitcoin Market Sentiment Analysis

## Project Overview

This project analyzes the relationship between **Bitcoin market sentiment** (Fear & Greed Index) and **Hyperliquid trader performance**.

The analysis combines Bitcoin Fear & Greed data with Hyperliquid trading records to identify how trader profitability and trading behaviour change across different market sentiment conditions.

---

## Objectives

- Clean and preprocess trading and sentiment datasets
- Merge datasets using trading date
- Perform Exploratory Data Analysis (EDA)
- Analyze trader profitability across market sentiments
- Compare BUY and SELL performance
- Generate business insights and recommendations

---

## Dataset

The project uses two datasets:

- Hyperliquid Trader Data
- Bitcoin Fear & Greed Index

## Key Findings

- Trading activity was highest during Fear.
- Average Closed PnL was highest during Extreme Greed.
- BUY trades performed better during Fear.
- SELL trades performed better during Greed and Extreme Greed.
- HYPE, SOL, ETH and BTC generated the highest realized profits among the identifiable assets.
